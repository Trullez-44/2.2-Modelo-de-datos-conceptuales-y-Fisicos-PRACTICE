# ***MY CONCEPTUAL DATA MODEL***

My entities to create an Entity-Relationship Model (ERM) are as follows:

1. **Product**:
    - Specifies the type of product to be carried.
    
    **Attributes** 
    
    1. ID
    2. Product Name
    3. Brand
    4. Price
    5. Category ID
2. **Customer**:
    - Customer information.
    
    **Attributes**
    
    1. ID
    2. Customer Name
    3. Customer Last Name
    4. Customer Address
    5. Customer Email
    6. Customer Phone
3. **Purchase**:
    - Represents a purchase made by the customer.
    
    **Attributes**
    
    1. ID
    2. Purchase Date
    3. Total
4. **Invoice**:
    - Purchase invoice.
    
    **Attributes**
    
    1. ID
    2. Invoice Number
    3. Invoice Date
    4. Invoice Total
5. **Categories**:
    - Categories of the products.
    
    **Attributes**
    
    1. ID
    2. Category Name
